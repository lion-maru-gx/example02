画面構成

以下の画面の遷移を行う

  • main(開始画面)
  • page1(開始画面より遷移)
  • page11(page1より遷移)
  • page2(開始画面より遷移)

Main.java

全ての画面をemumで定義して他のクラスからも参照できるようにpublc staticで定義する。

	public static enum Page{
		Main("main.fxml"),
		Page1("page1.fxml"),
		Page11("page11.fxml"),
		Page2("page2.fxml");

		// フィールドを定義
		private String url;

		// コンストラクタを定義
		private Page(String url) {
			this.url = url;
		}

		// メソッド
		public String getValue() {
			return this.url;
		}

	};
